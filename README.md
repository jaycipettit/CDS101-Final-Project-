# CDS 101 – Group Project Starter Kit (GMU, RMarkdown Version, v2)

This project is preconfigured for **RStudio**, **R Markdown**, and the **CDS 101 project rubric**.

Key files:
- `CDS101_Project.Rproj` – open this in RStudio
- `proposal_template.md` – proposal (dataset, RQ, models, expected results)
- `final_report_template.Rmd` – final report aligned with rubric
- `notebooks/eda_walkthrough.Rmd` – example EDA, explicitly mapped to rubric criteria
- `install_packages.R` – installs required packages (tidyverse, rmarkdown, ggplot2, etc.)
- `gmu-cds101.css` – GMU-branded styling for HTML output
- `Makefile` – optional command-line knitting

See inside each file for detailed instructions.
